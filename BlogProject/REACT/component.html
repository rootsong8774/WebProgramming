Component

React Element

  • JSX 문법으로 작성한 요소는 자바스크립트의 객체가 되고 이를 React element라고 부른다.
    •     
        import ReactDOM from 'react-dom';
        
        const element = <h1>안녕 리액트!</h1>;
        console.log(element);
        ReactDOM.render(element, document.getElementById('root'));
    

        
        {$$typeof: Symbol(react.element), type: "h1", key: null, ref: null, props: {…}, …}
  • 이 react element를 ReactDOM.render 함수의 argument로 전달하게 되면 react가 객체 형태의 값을 해석해서 HTML 형태로 브라우저에 출력한다.

React Component

  • React Component는 React Element를 더 자유롭게 다루기 위한 문법이다.
  • 다음과 같이 함수로 컴포넌트를 작성하면 Component의 함수명을 통해 하나의 태그처럼 사용할 수 있다.
    •     
    import ReactDOM from 'react-dom';
    
    function Hello() {
        return <h1>안녕 리액트</h1>;
    }
    
    const element = (
        <>
            <Hello />
            <Hello />
            <Hello />
        </>
    );
    
    ReactDOM.render(element, document.getElementById('root'));
  • 여기에 모듈을 활용하면 더 독립적으로 Component를 만들어서 활용할 수 있다.
    •     
    import diceBlue01 from './assets/dice-blue-1.svg';
    
    function Dice() {
        return <img src={diceBlue01} alt="Dice"/>
            
    }
    
    export default Dice;
    

        
    import Dice from './Dice';
    
    function App() {
        return (
            <div>
                <Dice />
            </div>
        );
    }
    
    export default App;
