WebStorm으로 React 시작하기

React에 대하여

  • VirtualDOM : React가 관리하는 가상 DOM. (브라우저의 DOM과 다름)
    • 페이지 내의 내용이 변경되면 VirtualDOM은 차이가 있는 부분을 확인한다.
    • 자체적으로 변경된 부분만 찾아서 Reflow(재렌더링) 한다.
  • JSX (JavaScript + XML) : JavaScript를 확장한 프로그래밍 언어.

설치

  1. Node.js 설치
  2. React 설치
    3.     
    C:\Users\user>npm install -g create-react-app
  3. WebStorm 설치

React 프로젝트 생성 및 실행

  1. New Project 생성 (왼쪽 Side bar의 React App 선택)
  2. 터미널에서 npm start 를 실행한다. (개발 모드 실행)
  3. package.json의 dependencis에 axios 버전을 추가해준다.
    4.     
    "dependencies": {
        "axios": "^0.25.0",
    ...
  4. 개발 모드 종료할 때는 터미널에서 Ctrl+C를 입력한다.
