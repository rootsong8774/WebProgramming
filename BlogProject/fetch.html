fetch 함수

  • fetch 함수는 브라우저에서만 기본으로 지원하는 기능이다. 따라서 Node.js를 통해서 실행하기 위해서는 외부 모듈을 사용해야한다.
    
        import fetch from "node-fetch";
        
        fetch('https://jsonplaceholder.typicode.com/users')
            .then((response) => response.text())
            .then(result => {console.log(result);});
  • 하지만 이렇게 import하려고 해도 “SyntaxError: Cannot use import statement outside a module in nodeJs Error”라는 에러가 발생하는데 이는 사용하려는 package가 모듈 타입으로 되어있지 않기 때문이다. 따라서 package.json 파일을 열어 다음과 같이 “type”: “module”을 추가해주도록 하자.
    
        ...
            "name": "webprogramming",
            "version": "1.0.0",
            "type": "module",
        ...
