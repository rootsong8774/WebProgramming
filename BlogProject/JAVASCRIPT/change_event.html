change event

  • text 타입 input 태그 내에서 enter 키를 감지하기 위해 다음과 같은 조건문을 사용하기 쉽다.
    
    <!DOCTYPE html>
    <html lang="ko">
    <head>
        <meta charset="UTF-8">
        <title>Enter Key</title>
    </head>
    
    <body>
        <div id="container">
            <input id="input" type="text" placeholder="단어를 입력하세요.">
        </div>
        <script src="index.js">/<script>
    </body>
    </html>
    

    
    const input = document.querySelector('#input');
    input.addEventListener('keydown',function(e){
        let inputWord = '';
        if(e.keyCode == 13) {
            inputWord = input.value;
            input.value = '';
        }
    })
  • 하지만 이럴 경우 input 태그 내에서 키보드 입력할 때마다 event를 발생하게 되고 이는 성능적인 측면에서 좋지 않다.
  • text 타입의 input 태그에서는 enter키를 누르는 것으로 change 이벤트를 발생시킬 수 있기 때문에 다음과 같이 작성하게 되면 성능적인 측면에서나 코드의 가독성 측면에서나 도움이 될 것이다.
    
    const input = document.querySelector('#input');
    input.addEventListener('change', function(){
        let inputWord = input.value;
        input.value = '';
        });
