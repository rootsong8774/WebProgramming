SpringBoot 시작하기

  • 기존에 생성한 Spring boot application repository를 제거하고 처음부터 다시 빌드업
    • Lombok과 관련한 에러 발생 : Lombok의 version을 1.18.20으로 설정하여 재빌드
    • - Build.gradle내의 라이브러리 설정 문제로 인해 Test Build가 원활하게 수행되지 않음 아래와 같이 junit 라이브러리의 버전을 명시해서 생긴 문제 따라서 다음과 같이 수정

    as-is

        
    testImplementation 'org.junit.jupiter:junit-jupiter-api:5.8.2'
    testImplementation 'org.junit.jupiter:junit-jupiter-engine:5.8.2'

    to-be

        
    testImplementation 'org.junit.jupiter:junit-jupiter-api'
    testImplementation 'org.junit.jupiter:junit-jupiter-engine'
  • 해당 교재가 현재와 2년에 가까운 시간적 간극이 있어 저자가 이에 대응하여 업데이트에 필요한 내용을 추가하였다.
    • 스프링 부트와 AWS로 혼자 구현하는 웹 서비스 최신 코드로 변경하기
