Static Content

  • 파일을 웹브라우저에 그대로 전달 하는 것을 정적 컨텐츠라고 한다. Spring boot는 정적 컨텐츠를 default로 제공한다.
  • 보통 resource 디렉토리에서 static 폴더 안에 있는 파일을 spring container가 찾아서 브라우저에 전달하는 방식이다.
    •     
    <!DOCTYPE html>
    <html lang="en">
    
    <head>
        <meta charset="UTF-8">
        <title>static content</title>
    </head>
    
    <body>
        정적 컨텐츠입니다.
    </body>
    
    </html>
  • static 폴더에 html 파일을 작성하여 생성하고 (여기서는 hello-static.html이라는 이름으로 생성하였다.) spring Application을 실행, localhost:8080/hello-static.html 로 접근하면 해당 html을 직접 브라우저에 띄워준다.
