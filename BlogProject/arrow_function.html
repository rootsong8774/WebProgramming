Arrow function 사용 시 주의사항

  • 실제 boolean이 아니지만 boolean처럼 평가되는 값
  • 리턴부분에 중괄호를 사용할 경우 return을 명시하지 않으면 아무것도 return 되지 않는다.
    
        fetch('https://www.google.co.kr/webhp?hl=ko')
            .then((response)=> { response.text() })
            .then((result)=>{console.log(result);});
  • 위의 예시처럼 중괄호로 닫아두게 되면 return값이 존재하지 않기 때문에 콘솔창에 undefined로 표시된다. 따라서 이러한 문제가 없게 하기 위해서는 코드를 다음과 같이 수정해야 한다.
    
        fetch('https://www.google.co.kr/webhp?hl=ko')
            .then((response)=> response.text())
            .then((result)=>{console.log(result);});
